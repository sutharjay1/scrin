### Prerequisites

- Node.js (v14 or newer)
- TypeScript
- Expo CLI

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/sutharjay1/scrin.git
   cd scrin
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the Expo development server:

   ```bash
   npx expo start -c
   ```
